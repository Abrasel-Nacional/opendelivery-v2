!!! tip "Guide + contract"
    This page is the **OpenAPI contract** (endpoints, fields, errors, and examples) — enough to implement Indoor.

    If you are new to the domain (account, pre-close, channels, roles), read the [Indoor guide](../protocol/indoor.md) first.

    **Prerequisite:** [Orders](orders.md) (`orderType: INDOOR`). Indoor is an extension of Orders, not a standalone capability.

    **Contract language:** OpenAPI is **always English**. Protocol guides are bilingual (PT/EN).
